流程控制语句
    顺序结构语句 : 代码从上往下依次运行
    分支结构语句 : 选择性执行代码
        1. if 语句
            第一种格式
            if(条件判断){
                语句体;
            }

            第二种格式
             if(条件判断){
                语句体1;
             } else {
                语句体2;
             }

             第三种格式
              if(条件判断1){
                 语句体1;
              } else if(条件判断2) {
                 语句体2;
              } else if(条件判断2) {
                 语句体2;
              }...else{
                 语句体 n;
              }

        2. switch 语句
            注意事项 :
            1. 小括号类型 : byte,short,char,int,String,枚举
            2. case 后面的值不能重复
            3. case 后面的值只能是字面量
            4. break 不写会出现穿透现象
            switch(变量/表达式){
                case 值1:
                    语句体1;
                    break;
                case 值2:
                    语句体2;
                    break;
                case 值3:
                    语句体3;
                    break;
                ....
                default:
                    语句体 n;
            }
    范围性判断 : if
    几个固定的取值判断 : switch


    循环结构语句 : 代码可以反复执行
        1. while 循环
            初始化语句;
            while(条件判断){
                循环体语句;
                条件控制语句;
            }

        2. for 循环
            for( 初始化语句 ;条件判断 ; 条件控制语句){
                循环体语句
            }

        3. do...while循环
