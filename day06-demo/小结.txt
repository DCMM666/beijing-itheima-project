面相对象三大特征
    1. 封装

    2. 继承
        概述 : 类与类一种关系 , 可以理解子父类的关系 , 使用关键字 extends 实现继承
        什么时候使用 : 多个类中存在了共性 , 可以使用父类抽取共性内容, 让子类去继承父类
        优势 : 1. 提高代码的复用性  2. 提高代码的可维护性
        缺点 : 耦合性太高      开发原则: 高内聚,低耦合

        格式
         class 子类名 extends 父类名{  }
        注意 : 子类只能使用父类的非私有成员

        成员访问特点
            1. 成员变量 : 就近原则
            2. 成员方法 : 就近原则
            3. 构造方法
        继承中的方法重写
            概述 : 子类和父类出现了一模一样的方法(方法名 , 形参列表)
            什么时候使用 : 父类的功能满足不了子类时候, 子类需要把父类的方法进行重写
            注意事项 :
                1. 私有的方法不能重写
                2. @Override 注解 : 修饰方法,此注解用来校验一个方法是否属于重写的方法 , 不属于的话会在编译时期报错!!!`
                3. public > protected > 默认的 > private  : 子类重写的方法权限必须大于等于父类的方法的权限 , 一般保持一致!
                4. 子类重写的方法返回值类型可以是父类方法返回值类型的子类型 ,  一般保持一致!
        Object 类 : java 中所有的类都会间接或者直接继承 Object类
            public String toString()
            public boolean equals(Object o)
        结果 :
            我们自己定义的类 如果没有重写 toString 方法 , 打印对象打印的是地址值 , 如果重写了 toString 打印的是对象内容!
            我们自己定义的类 如果没有重写 equals 方法 , 比较对象比较的是地址值是否相等 , 如果重写了 equals 比较的是对象内容是否相等!

        Java 的继承特性 :
            1. Java 中类与类属于单继承
            2. Java 中类与类不能多继承 , 但是可以多层继承
